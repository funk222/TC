# Changelog

## v0.2.0 - 2026-02-15

### 新增
- 新增主机同步接口 `GET /api/sync/status`，用于副机状态镜像显示。
- 新增副机固件（ESP8266 HW-364A）：`src/receiver_main.cpp`。
- 新增副机说明文档：`docs/ESP32_WIFI_SYNC.md`。

### 改进
- 支持同仓库双目标构建：
  - 主机：`esp32-c6-devkitc-1`
  - 副机：`esp8266-hw-364a`
- 副机显示优化：
  - 第一行大字闪烁告警/链路错误
  - 第二行固定 `T1/T2` 标签
  - 温度数值大字体显示
  - 底部同一行显示 `Heat` 与 `Sys`
  - 告警文本支持 `Too Low` / `Too High` / `dT>10C`
  - 闪烁频率提升（更均匀）

### 交互
- 主机菜单交互改为仅编码器按键控制：
  - 短按确认
  - 长按进入菜单/返回上一级
